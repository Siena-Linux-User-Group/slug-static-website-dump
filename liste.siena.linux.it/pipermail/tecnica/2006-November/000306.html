RTFM: https://help.ubuntu.com/community/EdgyUpgrades

Problems with X

If after the upgrade the X server doesn't start (i.e. if you can't get the 
graphic login), then it's because the xserver-xorg has been removed.

Reinstall it issuing:

  sudo apt-get install xserver-xorg 

Sometimes xserver-xorg needs to be reconfigured for your graphics card:

  sudo dpkg-reconfigure xserver-xorg 

Ci altri consigli sull'argomento.
