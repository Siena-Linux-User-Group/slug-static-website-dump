Rispondo per il punto due perché in realtà lo puoi fare, ad una condizione.
Se vuoi rimuovere dal repository un file che hai messo in gitignore (cosa
che credo sia il tuo caso) puoi fare

git rm --cached myfile


oppure, nel caso di cartella

git rm --cached -r mydirectory

Così facendo rimuovi il file dal repository, ma dovrebbe rimanere sul tuo
filesystem.
