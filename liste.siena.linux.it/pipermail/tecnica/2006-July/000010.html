Ciao Jacopo,

tempo addietro ho migrato un mailman da un server all'altro e copiando le 
directory ed ha funzionato tutto.

Per trovare dove mettere la roba (siccome anche io non lo sapevo proprio) 
creai una lista con un nome strano (cippalippa) nel server nuovo e sul 
vecchio e poi con un bel find trovai le directory, che naturalmente non mi 
ricordo più, sennò te lo scrivevo. :D

L'unico problema che ho incontrato é che per ogni lista c'e' una directory 
lock, e se dentro c'e' un file vuoto con il pid del qrunner, un altro qrunner 
con pid diverso non la scoda. Se hai un backup magari in quel momento stava 
girando il qrunner su qualche lista, quindi non ti funzionerebbe facendo il 
restore. Basta cancellare il file e riparte alla grande.

Ciao
