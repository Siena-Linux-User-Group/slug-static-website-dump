I server che abbiamo in giro per internet, come Gaia per intendersi, hanno:

SSH
IPTABLE configurato (chiaramente)
POSTFIX
BIND
MAILMAN
POSTGRESS
MYSQL
APACHE + PHP + ModPython + Mod_tutto_quello_che_serve + decine di CMS del 
cavolo in PHP
CVS con ViewCVS in web

Sono macchine virtuali User Mod Linux che girano con 128 mega di Ram, si avete 
capito bene: CENTOVENTOTTOMEGADIRAM!

In effetti quando abbiamo messo SpamAssassin la Serverina ha avuto qualche 
problema, gaia meno perché non ha Postgress SQL.

Un Pentium 2 é una bomba a confronto, se poi vuoi fare solo quello che dici 
vai anche con 64 Mega. ;)
