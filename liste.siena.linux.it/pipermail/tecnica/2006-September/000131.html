On Monday 18 September 2006 02:13, Yusef Maali wrote:
> Esiste un software che mi permetta di scaricare del materiale via ftp
> in modo incrementale?

Non credo. Sai perchè ? L'FTP è un protocollo che ti permette di fare 
upload/download dei file: il protocollo in se non è dato conoscere nè che 
file siano nè che contenuti abbiano. Al massimo puoi fare il download dei 
file che ti "mancano", una specie di "rsync" su una o più cartelle.

> Per incrementale intendo che mi dia la possibilità di scaricare via
> via solo i contenuti nuovi o modificati senza farmi riscaricare tutto
> (come file immutati).

Mmm...per differenziale intendi i file modificati o il contenuto modificato 
dei file modificati (scusa il gioco di parole) ?
Se intendevi la prima, qualcosa esiste. Alle bruttte "python+twisted" e ti 
scrivi un client ftp da solo in pochi minuti :-D
