Ciao a tutti,
ho installato, per esigenze di lavoro, logrotate su un HPUX. Questa la 
configurazione (/etc/logrotate.conf):

weekly
rotate 24
errors root
create
compress
/xxx/apache/logs/stats_log {
        daily
        rotate 31
        create
        compress
        postrotate
                /xxx/apache/bin/apachectl restart
        endscript
}

/xxx/apache/logs/access_log {
        weekly
        rotate 24
        create
        compress
        postrotate
                /xxx/apache/bin/apachectl restart
        endscript
}

e poi ho messo una entry nel cron. Tutto funziona, se non che 
logrotate "ruota" solo una volta ! Invece di ruotare, come nel primo caso, 31 
volte e 24 nel secondo, fà solo una rotazione, cancellando il file vecchio 
dei log. Ho controllato anche il file "status" (var/lob/logrotate/status):

sbs3:/xxxx/apache/logs# cat /var/lib/logrotate/status
logrotate state -- version 1
/xxx/apache/logs/stats_log 2007-4-26
/xxx/apache/logs/access_log 2007-4-24

idee ? Su google non ho trovato nulla :-(
