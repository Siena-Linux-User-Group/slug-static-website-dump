Non è possibile che ogni volta che non funziona un'interfaccia in questa lista si 
consigli di far finta di niente e andare ad editare a mano gli stessi file che avrebbe 
editato l'interfaccia.

Quell'interfaccia peraltro fa qualcosa di più, gestisce configurazioni multiple, 
eccezioni, ecc, cose che invece editando i file devi gestirti di nuovo *a mano*.

Se c'è un problema, per il suo bene, deve capire che problema è ed imparare come si 
risolve. Sta facendo advocacy all'università di Siena, non sta configuando il pc al 
nonno.

E se è un baco allora si segnala nel bug tracker. Altrimenti il software rimane quello 
che è proprio per colpa nostra, per giunta.

Per provare a diagnosticare il problema sul network manager ci sono delle istruzioni sul 
wiki:

https://wiki.ubuntu.com/DebuggingNetworkManager

Ti ripeto che a me funziona perfettamente sui miei 3 pc di cas, quindi o dipende 
dall'hardware oppure c'è qualche file che l'interfaccia non riesce a scrivere in quei 
pc.
