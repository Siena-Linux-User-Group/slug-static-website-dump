Ciao
se vuoi vivere felice lascia perdere il Network Manager e apri una shell:

$ sudo -s

# cd /etc/network/
# vi interfaces

... e da qui configuri tutte le interfacce che vuoi.
Es:
--------------------------------------------------------------

# This file describes the network interfaces available on your system
# and how to activate them. For more information, see interfaces(5).

# The loopback network interface
auto lo
iface lo inet loopback

# The primary network interface
auto eth0
#iface eth0 inet dhcp
iface eth0 inet static
address 192.168.7.106
netmask 255.255.255.0
network 192.168.7.0
broadcast 192.168.7.255
gateway 192.168.7.254
------------------------------------------------------

Come vedi, la eth0 prima era in dhcp e dopo e' con l'IP statico.

Per i DNS:

# cd /etc
# vi resolv.conf

nameserver
nameserver
nameserver
domain
search

Salvi e hai configurato tutto con l'IP statico (senno' il resolv.conf te lo scrive automaticamente il DHCP).


Buon navigazione.


PS: Se sapere quali IP liberi ci sono nella rete e che quindi puoi usare... o te lo dice un amministratore di sistema... o li cerchi con un programma tipo AngryIP (win32) e speri che non siano assegnati ma momentaneamente spenti.
