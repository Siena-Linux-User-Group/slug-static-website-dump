Ciao

Devi creare un dominio all'interno di plesk, esempio: ilmiodominio.it
e modificare il file /etc/hosts in locale per aggiungere
xxx.xxx.xxx.xxx ilmiodominio.it
Dopo questo puoi mettere i file di joomla in
/var/www/vhosts/ilmiodominio.it/httpdocs/test/

m.
