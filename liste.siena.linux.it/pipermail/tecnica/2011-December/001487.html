Chiedo scusa per il ritardo, non ero online nell'ultima settimana

Se hai creato il hosting del miodimonio.it,plesk automaticamente crea
la cartella /var/www/vhosts/ilmiodominio.it
all'interno trov 2 cartelle importanti:
* cartella httpdocs: Qui puoi inserire i file del tuo sito o
text.html, ci sono i file predefiniti del template predefinito di
plesk, puoi cancellare tutto
* cartella conf qui trovi il file httpd.include di plesk, non ti
consiglio di modificarlo,  perché plesk lo resetta ogni volta che fai
restart del server, ma puoi creare un file vhost.conf per modificare
le configurazione del tuo dominio, per esempoi cambiare il
open_basedir di php e modificare il directory index di apache e fare
un rewrite

per poter visualizzare ilmiodominio.it dal tuo browser, dovresti
modificare il file /etc/hosts del tuo pc, ed inserire xxx.xxx.xxx.xxx
ilmiodominio.it e xxx.xxx.xxx.xxx www.ilmiodominio.it

L'ho fatto tanti volte per provare siti in sviluppo o altri siti in
attesa di registrazione
