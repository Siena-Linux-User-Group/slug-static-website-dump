Nel file xorg.conf non hai la sezione "ServerLayout" definita, non mi
suona bene.

Intanto prova ad installarti i driver ati forniti dalla ati stessa, li
trovi a questo indirizzo:

    http://ati.amd.com/support/driver.html

Scarica ed installa (da root) il file .bin che ti danno.
Per riprova, da root prova a caricare il modulo in questo modo:

    modprobe fglrx

Se non ti restituisce nessun messaggio di errore, assicurati di
trovare il modulo fglrx nell'output fornito da:

    lsmod

Una volta fatto questo, usa un file di configurazione di questo tipo:

    http://www.pastebin.ca/1243477

Se non ci sono problemi strani e se scarichi i driver giusti, dovrebbe partire.

Sembra un "banale" problema di configurazione.
