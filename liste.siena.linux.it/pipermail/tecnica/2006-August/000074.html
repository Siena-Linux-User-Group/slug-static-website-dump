Il file xorg.conf è il file di configurazione del server X...

lo puoi editare tranquillamente anche con un editor grafico (kwrite,
kate...) a patto che tu apra tale programma come root:
- apri Konsole
- digita: su [enter] e poi inserisci la password di root
- digita: kwrite /etc/X11/xorg.conf (se xorg.conf non è lì fai un
updatedb e poi locate xorg.conf, sempre da root)

in seguito dentro al file xorg.conf hai una sezione nominata: InputDevice
dentro questa sezione c'è o ci potrebbe (nn è indispensabile) essere
una opzione: AccelFactor per darti un ordine di grandezza la mia è
settata a 0.0010

Section "InputDevice"
 Identifier      "blablabla"
 Driver  "blablabla"
 ...blablabla...
 Option "AccelFactor" "0.0010"
EndSection

ogni qual volta modifichi questo file devi poi riavviare X
