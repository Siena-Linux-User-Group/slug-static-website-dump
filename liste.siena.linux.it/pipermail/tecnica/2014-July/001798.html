Bisogna iniziare da

# dpkg --list
# cat /etc/apt/sources.list

giuso per avere un idea di cosa hai di installato


in generale facendo

# apt-get update && apt-get upgrade

modifica /etc/apt/sources.list, canbiabno "squeeze" a "wheezy", e di nuovo
dopo

# apt-get update && apt-get upgrade && apt-get dist-upgrade


dovrebbe andare bene, consigliato fare backup alla cartella /etc
