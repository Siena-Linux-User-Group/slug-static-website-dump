Ciao, non sono un esperto di Gaim perché utilizzo Kopete, però la
configurazione dovrebbe essere la stessa.

Io ho impostato lo stesso server MSN (messenger.hotmail.com) sulla tua
stessa porta (1863) e mi funziona.
Magari se non hai un proxy digli che non c'è un proxy invece di
utilizzare le impostazioni globali.

Se invece c'è un proxy dillo esplicitamente (e usa HTTP se la porta
1863 non è aperta)

Sennò prova ad autenticarti sul web messenger della Microsoft
http://webmessenger.msn.com/
giusto per verificare che user e pwd siano corrette (funziona anche se
il browser non è supportato, ti lascio immaginare quale sia l'unico
supportato...)
