Salvare solo i file di configurazioni non ti basta.
Salvare le configurazione del kernel non ti basta.
Devi salvare tutto.

Ti descrivo un approccio abbastanza gentooista:
Fai il backup di tutto con:
sudo tar cjf /backup.tar.bz2 / --exclude=/tmp --exclude=/mnt --exclude=/proc --exclude=/sys

Per ripristinare tutto (magari attraverso una live):
sudo tar -xvjpf backup.tar.bz2


Detto questo ti do la mia assoluta certezza che linux non funzionerà subito automaticamente.
Avrai comunque bisogno di perderci tempo dietro, magari meno che reinstallare tutto da zero.

Se cerchi tool automatici che tengano conto del cambio di partizione, non so aiutarti.
