Scusa un attimo, ma in questo link ci sono i sorgenti ma anche i pacchetti Deb 
per Ubuntu, perché non usi quelli?

Comunque, per compilare un programma da sorgenti non c'è una regola precisa. O 
meglio si: apri il pacchetto e dentro trovi sempre un file che spiega cosa si 
deve fare chiamato o README o INSTALL o a volte  entrambi, lo apri e lo leggi 
:)

Se poi c'è da compilare con i tools autoconf del progetto GNU la sequenza è

$ ./configure

elabora un pò dopodiché

$ make

e alla fine, per installare nel sistema

$ sudo make install

ma per compilare i programmi la prima cosa è leggere la documentazione 
allegata.

Facci sapere se incontri problemi,
