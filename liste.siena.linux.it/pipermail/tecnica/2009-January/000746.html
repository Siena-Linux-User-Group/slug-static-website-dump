In ogni caso per compilare con MAKE  di solitodevi decomprimere tutti i
sorgenti, poi fare cd nella directory che hai creato e infine digitare
make
make install
Insieme ai sorgenti c'e' di solito un file di testo che spiega come
compilare. E' bene leggerlo perché potrebbe contenere informazioni utili. Di
solito si chiama INSTALL o INSTALL.TXT.
