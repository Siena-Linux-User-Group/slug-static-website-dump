ciao,
ho scaricato il pacchetto e dentro vi ho trovato un file in java e un
comodo lanciatore ".sh".
Ti conviene quindi decomprimere il file dentro una cartella, entrarvi
e, se hai ubuntu, premere sul file "startJTrak.sh" e premere su
esegui. Se questo non fosse possibile/non funzionasse lo avvii da
shell spostandoti nella cartella e scriveno "sh statJTrak.sh" o
"./startJTrak.sh" che è equivalente.
Se queste operazioni le hai già svolte senza successo probabilmente
non hai installato il java, quindi devi provvedere ad installare
"sun-java6-jre" (se hai una distribuzione diversa da una debian/*buntu
o parente, il nome potrebbe cambiare).

Se hai altri problemi fammi sapere.
Ciao.
