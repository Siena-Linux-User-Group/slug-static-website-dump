> *B1*) Posso programmare in qualche modo un dump automatico del database
> SQL di Drupal? Ho trovato dei moduli per drupal che dovrebbero fare questo
> ma non funzionano come vorrei. In particolar modo vorrei che venissero
> salvati dei file di backup (del tipo *.sql) ad intervalli regolari (solo del
> database di drupal). Sapete darmi qualche consiglio? Sapevo che in linux si
> possono fare degli script che vengono eseguiti in automatico dalla shell ma
> non so né come farli né come funzionano.
>

devi usare una utility di backup tipo mysql_dump (per MySQL) o pg_dump
(PostgreSQL)
una volta che hai trovato i parametri corretti per l'esecuzione la scheduli
con:

  $ sudo crontab -e -u <utente-per-il-backup>

dove <utente-per-il-backup> è lo user che vuoi che esegua i backup
