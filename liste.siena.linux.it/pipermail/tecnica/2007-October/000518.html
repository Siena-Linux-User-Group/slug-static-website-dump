Quindi per "creare una rete" intendi condividere i file?
Cioè SMB e NFS sono protocolli per la condivisione di file in rete, mentre per
quanto riguarda il fatto cui ti riferivi, al pingare gli ip, quelli
dovresti pingarli a prescindere
dal fatto che samba o altri siano configurati correttamente.

Questo è quello che uso io nel mio file di samba:

[global]
# Nome del Workgroup ^^
workgroup = Workgroup
# Identificativo della Macchina
server string = Michele's Laptop
# Tipo di restrizioni usate
security = share
# Hosts da cui accettare connessioni
hosts allow = 192.168.1. 127.

[Condivisione]
        comment = File Condivisi
        path = /home/mithenks/condivisi
        public = yes
        writeable = no
        guest ok = yes

Prova ^^
