Ok, ho capito:
Prima di tutto lascia perdere l'interfaccia grafica e apri un bel terminale.
La prima cosa è capire se la rete funziona con il comando
ping indirizzo
dove indirizzo è l'IP dell'altro computer.

Se hai il reply la rete funziona. Altrimenti devi sistemare prima questa.

Se la rete funziona si prova samba.
Edita il file di configurazione, leggi bene il mauale in italiano riguardo a questo file. Usa all'inizio l'opzione SHARE.
Poi esegui il comando testparm. Serve a vedere se il file di conf. è a posto.

Dopo di che sempre nel terminale usa l'utility smbclient che si collega sia all'altro server samba o anche a quello che è sulla macchina stessa.

Leggi il manuale di smbclient su internet oppure con il comando 
man smbclient.
