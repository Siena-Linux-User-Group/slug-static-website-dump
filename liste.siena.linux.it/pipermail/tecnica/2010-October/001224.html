non avevo inteso rsync, ...

supponiamo tu debba ottenere un rc pari a 0 ovvero tutti i file trasferiti


RC=1
while [[ RC -ne 0 ]]
do
rsync -a etc.etc.etc.etc.
RC=$?
done

attenzione, che in caso di problemi di permessi cio' portera' a cicli
infiniti. Quindi puoi inserirci qualche controllo
gli exit code di rsync sono qui.. non so se aggiornato (googlato)

http://wpkg.org/Rsync_exit_codes

un controllo potrebbe essere

RC=12; while [[ $RC -eq 12 || $RC -eq 30 ]]
