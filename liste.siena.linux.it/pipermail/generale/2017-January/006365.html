Ciao a tutti,

la ditta per cui lavoro (LdP Progetti GIS) sta cercando uno sviluppatore
Web e/o Android con le seguenti caratteristiche:

   - buona conoscenza di PHP, PostgreSQL e Debian/Centos
   - buona conoscenza degli strumenti di sviluppo Android e di Java
   - conoscenza di GIT o altri strumenti di versioning del codice
   - attitudine a lavorare in gruppo e a scrivere codice manutenibile


Requisiti preferenziali ma non indispensabili:

   - conoscenza di shell scripting
   - conoscenze di cartografia digitale
   - conoscenza di QGIS o altri strumenti GIS desktop
   - competenze sistemistiche di base in ambiente GNU/Linux e Windows

Disponibilità immediata, orario full-time, sede di lavoro Siena.

Viene offerto un contratto a tempo determinato di qualche mese per prova.
Se la prova è soddisfacente verrà proposto un contratto a tempo
indeterminato.
Il contratto collettivo di riferimento è CCNL Studi Professionali.

Se siete interessati o conoscete qualcuno di valido che possa essere
interessato, inviatemi un CV all'indirizzo monfardini a ldpgis.it

Cordiali saluti,
