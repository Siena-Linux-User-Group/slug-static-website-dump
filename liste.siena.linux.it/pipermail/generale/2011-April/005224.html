Il PHP e' concepito per essere interpretato ad ogni interrogazione.
Autonomamente non fa niente se non viene processato da Apache. Quindi
non ho ben capito cosa stai cercando di fare. Se vuoi aggiornare dei
contenuti che poi vengono mostrati su una pagina web quando un utente
si collega, devi implementare un sistema dedicato nel server che in
qualche modo memorizzi i risultati in un db piuttosto che su un file
(per esempio la programmazione di un cinema) ed utilizzare queste info
a livello di codice php per essere visualizzate.
