Luca,

il vecchio SAMBA non ti piace più ? Fai un bel serverino linux con
SAMBA, directory "Share" condivisa in scrittura e lettura e via. Da Web
puoi condividere gli stessi file, puntando (ad esempio) la root del
server apache2 nella medesima cartella.

Se questa soluzione non te gusta, prova un web based file manager in
PHP:
http://www.nilambar.net/2012/10/best-free-web-based-file-management-scripts-in-php.html
