Alle 12:53, giovedì 16 novembre 2006, Freeze North Pole ha scritto:
> /usr/lib/xorg/modules/extensions/libGLcore.sd

1- Controlla se questo file esiste.
 
2- Guarda nei file di log, tipo /var/log/Xorg.0.log. Alla fine dovrebbe 
esserci indicato il motivo dettagliato del crash. Controlla l'errore e 
postacelo.

3- Postaci la Section "Module" del file /etc/X11/xorg.conf. Dovrebbe esserci 
un load "glcore". Prova a commentarlo:
#    Load           "glcore"
Oppure prova a cavare Load "glx" (la sparo a caso).

NOTA: prima di modificare il file xorg.conf, crea un backup
cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup
