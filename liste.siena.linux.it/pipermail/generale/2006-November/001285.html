Il JavaDay (http://roma.javaday.it) è una manifestazione alla sua prima
edizione, promossa dai Java User Group italiani, dalla community di
JavaPortal e dal JIA.

Il programma del JavaDay romano prevede 16 talk, organizzati in quattro
tracce per diverse tipologie di pubblico:
- Java Junior
- Java Developer
- Java Architect
- Java Open Source

Gli argomenti trattati nei 16 talk sono particolarmente attenti alle
novità del panorama java e alle necessità che possono incontrare gli
sviluppatori.

Infatti ci saranno talk tecnici sui framework e tecnologie attualmente
più utilizzati come Spring, Maven, SOA, Ajax ma anche su framework
emergenti come Seam, oppure talk di approfondimento su Java base, UML,
teoria delle programmazione ad oggetti.

In più, si affronterà anche il tema del recente rilascio di Java in open
source.

Per visionare il programma completo e registrarsi al JavaDay romano
consultate il seguente link: http://roma.javaday.it

La partecipazione è gratuita, è gradita la prenotazione.
