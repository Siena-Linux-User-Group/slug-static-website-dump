Iniziamo dal principio...
Dato che  lo lanci con ./SweetHome3D mi pare di capire che questo file
sia un eseguibile e non una classe java!

Se e' uno script di shell dovresti allegarmelo in modo che possa capire cosa fa.

Se Invece funziona quando fai:

cd /home/robi/SweetHome3D-1.2.1
java SweetHome3D

allora e' java.

Se funziona cosi' perche' non fare uno script:

----------------------------------------------------

#!/bin/bash

DIR=$PWD
cd /home/robi/SweetHome3D-1.2.1
java SweetHome3D
cd $DIR

----------------------------------------------------

Con $MAINCLASS intendevo la classe principale java quella che contiene
il metodo main() da  eseguire.
Con $CLASSPATH intendevo l'inseme di PATH separati da : contenenti
tutte le classi java o archivi .jar necessari all'esecuzione del
metodo main() della classe principale.

Proseguiamo in pvt nel caso, non credo sia argomento di interesse "generale".
