Per accattivarmi adesioni, un esempietto:

Come si scrive una funzione f(x) che restituisce:
3000 se x == 3
-1      se x*x = x*2
x*x    se x è pari
0       altrimenti

In haskell si scrive così, chi prova a mandarla in un linguaggio in cui
"viene meglio"?

f 3 = 3000
f x | x*x == x*2 = -1
f x | even x     = x^2
f x | otherwise  = 0

per provarla:

map f [0..10]
stampa [0,-1,3000,16,0,36,0,64,0,100]
