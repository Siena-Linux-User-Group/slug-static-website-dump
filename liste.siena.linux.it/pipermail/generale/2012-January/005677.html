2012/1/26 Alessandro Iurlano <alessandro.iurlano a gmail.com>:
> Se stai usando bash queste impostazioni le trovi solitamente nel file
> .bashrc nella home dell'utente.
> Nel mio sistema ho il file /etc/skel/.bashrc che ha le impostazioni
> sia per il prompt che per i colori di ls.
> Prova:
>
> $ cp .bashrc bashrc-backup
> $ cp /etc/skel/.bashrc .bashrc
> $ bash

Bhe in realtà se ha creato un utente ex novo quei file da skel sono
già stati copiati.

Potresti avere un .bashrc dentro la tua home fatto così
if [ -f /etc/bashrc ]; then
	. /etc/bashrc
fi
che te lo fa puntare a /etc/bashrc globale, vedi cosa contiene.
Se come terminale hai xterm (lo vedi in $TERM) e vuoi un prompt carino
potresti inserire questo dentro il tuo .bashrc
PROMPT_COMMAND='printf "\033]0;%s@%s:%s\007" "${USER}"
"${HOSTNAME%%.*}" "${PWD/#$HOME/~}"'
